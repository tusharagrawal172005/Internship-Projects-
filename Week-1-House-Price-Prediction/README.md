# 🏠 House Price Prediction

## 📌 Project Overview

This project was completed as part of my **Data Analytics Internship (Week 1)**.

The objective of this project was to build machine learning models capable of predicting house prices using different property features. It covers the complete machine learning workflow, including data exploration, preprocessing, visualization, model training, evaluation, and interpretation of results.

---

## 🎯 Problem Statement

House prices depend on several factors such as area, number of bedrooms, bathrooms, parking availability, furnishing status, and other amenities.

The goal of this project is to predict the selling price of a house using these features and identify which factors influence house prices the most.

---

## 📌 Results

- Area was the most influential feature affecting house prices.
- Bathrooms, air conditioning, and number of stories also had a significant impact.
- Linear Regression achieved an **R² Score of approximately 0.65**.
- Linear Regression slightly outperformed Random Forest on this dataset.
